## Key v2026.3.28 Changes by Domain

### Breaking Changes
- Providers/Qwen: remove deprecated `qwen-portal-auth` OAuth integration
- Config/Doctor: drop automatic config migrations older than two months

### New Features
- xAI: Responses API, plugin-owned x_search, auto-enable
- MiniMax: image-01 generation provider; catalog trimmed to M2.7 only
- Plugins/hooks: async `requireApproval` in `before_tool_call` hooks
- ACP/channels: current-conversation binds for Discord, BlueBubbles, iMessage
- OpenAI: `apply_patch` enabled by default for OpenAI/Codex
- Plugins/CLI backends: bundled Claude CLI, Codex CLI, Gemini CLI on plugin surface
- Podman: simplified rootless user setup
- Slack: new `upload-file` action; Teams/Google Chat/BlueBubbles file unification
- Matrix TTS: native voice bubbles instead of generic audio
- CLI: `openclaw config schema` command
- Config/TTS: auto-migrate legacy speech config
- Memory/plugins: pre-compaction flush behind memory plugin contract
- MCP/channels: Gateway-backed channel MCP bridge with Codex/Claude tools

### Key Fixes
- Memory/QMD: CJK chunking, surrogate-pair fix, embedInterval fix, slugified paths
- Memory flush: append-only daily files
- Agents: unhandled stop reasons, per-model cooldowns, compaction improvements
- Telegram: forum topic routing, proportional split fix, whitespace-only reply skip
- WhatsApp: infinite echo loop in self-chat DM
- Discord: reconnect drain stale sockets
- Matrix: E2EE thumbnail encryption, ESM crypto-nodejs fix
- LINE: timing-safe HMAC, status snapshot config fix
- Teams: Entra JWT fallback, config schema fixes
- BlueBubbles: debounce null guard, inbound image refs
- iMessage: inline reply_to tag leak
- Auto-reply: JSON envelope suppression
- CLI/zsh: defer compdef registration
- Security/audit: web search key audit expansion
