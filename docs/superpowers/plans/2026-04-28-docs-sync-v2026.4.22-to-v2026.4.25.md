## Source Material Reference

The following raw changelog data was extracted from the openclaw source at the respective git tags. Use this as the authoritative source for all doc synthesis.

### v2026.4.22 — Key Changes
- **Providers/xAI:** image gen (`grok-imagine-image`/`-pro`), TTS (6 voices, MP3/WAV/PCM/G.711), STT (`grok-stt`), realtime Voice Call transcription
- **Providers/STT:** Voice Call streaming for Deepgram, ElevenLabs (+ Scribe v2 batch), Mistral
- **TUI:** local embedded mode (no Gateway, plugin approvals enforced)
- **Onboarding:** auto-install missing provider/channel plugins on first run
- **OpenAI/Responses:** native `web_search` tool auto-used for Responses models
- **Models/commands:** `/models add <provider> <modelId>` — register models from chat without restart
- **WhatsApp:** `replyToMode` native reply quoting; per-group/direct `systemPrompt` → `GroupSystemPrompt`
- **Agents/sessions:** `sessions_list` mailbox filters (label, agent, search) + derived title/preview
- **Control UI:** browser-local operator identity (name + avatar), Quick Settings tightening
- **Gateway/diagnostics:** stability recording by default; support diagnostics export
- **Providers/Tencent:** bundled Tencent Cloud plugin (`hy3-preview`, tiered pricing)
- **Providers/Amazon Bedrock Mantle:** Claude Opus 4.7 via Mantle Anthropic Messages route
- **Providers/GPT-5:** overlay moved to shared provider runtime; `agents.defaults.promptOverlays.gpt5.personality`
- **Providers/OpenAI Codex:** removed `~/.codex` OAuth import from onboarding
- **CLI/Claude:** warm stdio sessions by default; resume after Gateway restarts
- **Pi/models:** updated to `0.68.1`; OpenCode Go catalog from pi
- **Tokenjuice:** bundled opt-in plugin for compacting exec/bash results in Pi runs
- **ACPX:** explicit `openClawToolsMcpBridge` option for core MCP server injection (cron)
- **Docs/i18n:** Thai translation support
- **Plugin SDK/STT:** shared realtime WebSocket transport + multipart batch helpers
- **Codex harness/hooks:** `before_prompt_build`, `before_compaction`/`after_compaction`, `llm_input`/`llm_output`/`agent_end` for native Codex runs

### v2026.4.23 — Key Changes
- **Providers/OpenAI:** image gen + reference-image editing via Codex OAuth (no `OPENAI_API_KEY` needed)
- **Providers/OpenRouter:** image gen + reference-image editing via `image_generate`
- **Image generation:** agent-requestable quality/format hints; OpenAI background/moderation/compression hints
- **Agents/subagents:** optional forked context for `sessions_spawn` (child inherits requester transcript)
- **Agents/tools:** per-call `timeoutMs` for image/video/music/TTS tools
- **Memory/local embeddings:** configurable `memorySearch.local.contextSize` (default 4096)
- **Dependencies/Pi:** updated to `0.70.0`; upstream `gpt-5.5` catalog metadata
- **Codex harness:** structured debug logging for harness selection decisions
- **Security fixes:** ~30 security patches (exec approvals, cleartext pairing, SSRF, Teams JWT, owner-only bypass in cron, cleartext LAN gateway)
- **Memory/dreaming:** decoupled from heartbeat — dreaming runs as isolated lightweight agent turn; `openclaw doctor --fix` migrates stale dreaming cron jobs. Fixes #69811, #67397, #68972

### v2026.4.24 — Key Highlights
- **Google Meet:** bundled participant plugin (personal Google auth, Chrome/Twilio realtime sessions, paired-node Chrome, artifact/attendance exports)
- **DeepSeek V4 Flash + V4 Pro:** bundled catalog; V4 Flash is onboarding default
- **Voice/Talk/Google Meet:** realtime voice loops consulting full OpenClaw agent
- **Browser automation:** coordinate clicks, longer default action budgets, per-profile headless overrides
- **Plugin/model infrastructure:** static model catalogs, manifest-backed model rows, lazy provider deps, external runtime-dependency repair for packaged installs
- **Packaged installs:** preserve package-root runtime deps on Windows npm updates
- **Heartbeat:** clamp oversized scheduler delays via safe timer helper
- **Telegram:** remove startup persisted-offset `getUpdates` preflight

### v2026.4.25 — Key Highlights
- **TTS upgrade:** `/tts latest`, chat-scoped `auto-TTS`, personas, per-agent/per-account overrides; new providers: Azure Speech, Xiaomi (MiMo), Local CLI, Inworld, Volcengine/BytePlus Seed, ElevenLabs v3
- **Cold plugin registry:** plugin startup/install moves to cold persisted registry, cutting broad manifest scans; `openclaw plugins registry` CLI; `plugins.installs` no longer authored config
- **OpenTelemetry:** model calls, token usage, tool loops, harness runs, exec processes, outbound delivery, context assembly, memory pressure; bounded low-cardinality attributes; GenAI semantic conventions; Prometheus plugin (`diagnostics-prometheus`)
- **Browser automation:** safe tab URLs, iframe-aware role snapshots, CDP readiness tuning, headless one-shot launch, `openclaw browser doctor --deep`
- **Control UI:** PWA install + Web Push notifications; Crestodian first-run repair; TUI setup; context mode selector
- **Install/update hardening:** Windows, macOS, Linux, Docker, bundled plugin runtime deps, Node service restarts, LaunchAgent token rotation
- **Plugins/hooks:** before-agent-finalize hooks, cron `jobId` hook context, bounded native permission fingerprints, Codex MCP hook relay
- **Plugin startup:** move Gateway startup plugin planning onto versioned cold registry index
- **Tokenjuice:** bumped to 0.6.3
- **Android/Talk Mode:** Voice tab with runtime-owned voice capture modes
